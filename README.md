# < 출구 없는 밤 : 생존 프로토콜 >

## 1. 프로젝트 소개

< 출구 없는 밤 : 생존 프로토콜 >은 생성형 AI를 활용하여 개발한 Pygame 기반 미로 탈출 생존 전략 게임입니다.

플레이어는 제한 시간 안에 미로를 탐색하여 열쇠를 찾아 탈출해야 합니다. 게임 진행 중에는 적을 피하거나 공격하며 생존해야 하며, 다양한 난이도와 아이템/함정 기능을 통해 더욱 재미있게 플레이할 수 있습니다.

## 2. 개발 환경

* Python 3
* Pygame
* Pillow

## 3. 조작법

* ↑↓→← : 이동
* WASD : 일반 총알 발사 (적 처치)
* SPACE : 특수 총알 발사 (벽 파괴)

## 4. 주요 기능

### 1) 난이도 선택

* Easy : 120s / 적 1개
* Normal : 130s / 적 2개, 20초 마다 무한 생성
* Hard : 250s / 적 3개, 10초 마다 무한 생성
* 각 난이도 별로 5개의 STAGE 존재
* STAGE 2 부터는 열쇠를 찾아야 탈출구가 열림

### 2) 캐릭터 커스터마이즈

* 플레이어 모양 선택 (네모 / 세모 / 하트 / 별)
* 플레이어 색상 선택 (빨 / 주 / 노 / 초 / 파 / 보 / 핑 / 민 / 회 / 검)

### 3) 생존 및 전투 시스템

* 플레이어는 게임 시작 시 3개의 생명을 부여 받음
* 미로 내부의 적 or 함정과 접촉 시 생명 -1
* 미로 내부의 하트 접촉 시 생명 +1
* 총알 발사 기능 (일반 총알 : 적 처치 / 특수 총알 : 벽 파괴)

### 4) 게임 UI

* 메인 화면 (시작 / 캐릭터 커스텀 / 게임 설명)
* 시작 화면 → 난이도 선택 화면 → 스테이지 선택 화면
* 실행 화면 → 탈출 성공 / 실패 → 종료 화면

### 5) 구현 기능

* 실행 화면 상단 상태 바 : 생명 개수 / 총알 개수 / 경과 시간 / 남은 시간 / 난이도 및 스테이지
* 사운드 시스템 : bgm / 이벤트 별 효과음
* 게임 사용 데이터 저장 및 불러오기
* 난이도 조절 / 멀티 스테이지
* 캐릭터 커스터마이즈
* 외부 라이브러리 : Pygame / Pillow(PIL)

## 5. 실행 방법

1. 저장소를 클론합니다.

```bash
git clone <repository-url>
```

2. 필요한 라이브러리를 설치합니다.

```bash
pip install -r requirements.txt
```

3. 게임을 실행합니다.

```bash
MAIN.py
```
